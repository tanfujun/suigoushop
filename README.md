## suigoushop

随购商城后台管理系统



## 技术栈

vue2 + vuex + element-ui + axios + echarts + vue-router + vue-admin-template + scss



## 运行

git clone https://github.com/tanfujun/suigoushop.git

npm install 

npm run dev
